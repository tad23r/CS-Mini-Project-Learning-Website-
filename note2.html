Assignments & Operators

JavaScript provides many types of operators used to assign, compare, and perform arithmetic operations on values.

Assignment Operators

Assignment operators are used to assign values to variables:

  • = — Assign value: let x = 5;
  • += — Add and assign: x += 3; // x = x + 3
  • -= — Subtract and assign
  • *= — Multiply and assign
  • /= — Divide and assign
  • %= — Modulus and assign

Arithmetic Operators

Used to perform basic mathematical operations:

  • + — Addition
  • - — Subtraction
  • * — Multiplication
  • / — Division
  • % — Modulus (Remainder)
  • ** — Exponentiation

Comparison Operators

These compare two values and return a boolean:

  • == — Equal to (loose comparison)
  • === — Equal value and type (strict)
  • != — Not equal
  • !== — Not equal value or type
  • >, < — Greater or less than
  • >=, <= — Greater/less than or equal

Logical Operators

  • && — Logical AND
  • || — Logical OR
  • ! — Logical NOT
