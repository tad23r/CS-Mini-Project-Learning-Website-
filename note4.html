Functions

Functions are reusable blocks of code that perform specific tasks. They help keep your code organized and avoid repetition.

1. Declaring a Function


function greet(name) {
  console.log("Hello, " + name + "!");
}
greet("Alex"); // Hello, Alex!

2. Return Values


function add(a, b) {
  return a + b;
}
let sum = add(3, 4); // sum is 7

3. Function Expressions

You can also define functions using expressions:


const multiply = function(a, b) {
  return a * b;
};
console.log(multiply(2, 5)); // 10

4. Arrow Functions (ES6)

Arrow functions offer a shorter syntax:


const divide = (a, b) => a / b;
console.log(divide(10, 2)); // 5
