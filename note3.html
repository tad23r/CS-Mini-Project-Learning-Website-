Loops & Conditions

JavaScript allows you to control the flow of your code using conditions and loops. These are essential for making decisions and repeating actions.

1. Conditional Statements

Use conditions to perform actions based on certain checks:

  • if — Basic condition
  • else — Runs if if fails
  • else if — Extra condition checks
  • switch — Matches specific values

let age = 16;
if (age >= 18) {
  console.log("You're an adult!");
} else {
  console.log("You're still a teen.");
}

2. Loops

Loops are used to run the same block of code multiple times.

  • for — Runs a block a specific number of times
  • while — Repeats as long as a condition is true
  • do...while — Runs once, then repeats if true
  • for...of — Loops through values of an iterable (like arrays)
  • for...in — Loops through keys in an object

// Example: For loop
for (let i = 0; i < 5; i++) {
  console.log("Loop #" + i);
}

// Example: While loop
let x = 3;
while (x > 0) {
  console.log(x);
  x--;
}
