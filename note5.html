Objects & Arrays

Objects and arrays are two of the most important data structures in JavaScript.

1. JavaScript Objects

Objects store data in key-value pairs.


const person = {
  name: "Alice",
  age: 25,
  greet: function() {
    console.log("Hello!");
  }
};

console.log(person.name);     // Alice
person.greet();               // Hello!

2. JavaScript Arrays

Arrays store ordered lists of values.


const colors = ["red", "green", "blue"];
console.log(colors[0]);       // red
console.log(colors.length);   // 3

3. Array Methods


colors.push("yellow");        // adds to end
colors.pop();                 // removes last element
colors.shift();               // removes first element
colors.unshift("purple");     // adds to beginning

4. Combining Objects & Arrays


const people = [
  { name: "Tom", age: 20 },
  { name: "Jerry", age: 22 }
];

console.log(people[1].name);  // Jerry
